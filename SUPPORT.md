# Getting Help

Thank you for using a JBoss Logging project! Here are ways to get help:

## Documentation

- Check the project's README for documentation links
- Visit the individual repository for project-specific documentation

## Ask Questions

- **GitHub Discussions** - Ask questions in the repository's Discussions section (if enabled)
- **Stack Overflow** - Tag your questions with `jboss-logging` and the specific project name

## Report Bugs

- Most repositories use **GitHub Issues** for bug reports
- Some repositories use [JIRA](https://issues.redhat.com/projects/JBLOGGING)
- See [CONTRIBUTING.md](CONTRIBUTING.md) for details

## Security Issues

For security vulnerabilities, please see our [Security Policy](SECURITY.md).
